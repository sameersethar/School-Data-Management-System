# Student Data Management System
A console-based C++ application developed for my first-semester Programming Fundamentals project.

## 🚀 Features
- **Enter Data:** Collect student names, roll numbers, and contact info.
- **Show Data:** View all stored records in a formatted list.
- **Search:** Quickly find student records by Roll Number.
- **Update/Delete:** Modify or remove existing records.

## 🛠️ Tech Stack
- **Language:** C++
- **Concepts:** File Handling, Arrays, Input Validation, Custom Header Libraries.

## 📦 How to Run
1. Clone the repository.
2. Compile `PF Project .cpp` using any C++ compiler (e.g., MinGW, Dev-C++, or VS Code).
3. Run the generated executable.
